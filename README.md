# SQL Movie Analysis Project

This project analyzes Pixar movie data using SQL.

## Skills Used
- SELECT
- WHERE
- LIKE
- ORDER BY
- LIMIT
- GROUP BY
- Aggregate functions

## Dataset
Movies table containing title, director, release year, and duration.

## Tools
